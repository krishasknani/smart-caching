Generate Extension Icons

This page will help you generate the required PNG icons from the SVG.

Instructions:

  1. Open the SVG file (icon.svg) in a browser or image editor
  2. Export/save as PNG with these dimensions:
    • icon16.png - 16x16 pixels
    • icon48.png - 48x48 pixels
    • icon128.png - 128x128 pixels
  3. Save them in the icons/ directory

Alternative: Use this canvas to generate icons
